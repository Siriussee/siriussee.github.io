# Hey, Siri. 附近最近的宾馆在哪里？

## 0x00

我叫 SiriusSee ，他们都喜欢叫我 Siri 。作为一名 Agent ，我生平爱好之一就是躺在床上拉多重，直到闹钟响起，或是眼皮打架。

躺在床上拉多重是一件轻松惬意，又极度无聊的事情。我所要做的，无非就是三件事：用自己觉得最舒适的姿势躺好，打开 Scanner ，按顺序连接。仅此而已。

哦，对了，我还要去**隔壁学校** hack key 。

## 0x01

hack key 是另一件极度无聊的事情。它能带给我的唯一乐趣，大概只有绕圈所形成的轨迹了。

<img id="farm-track" src="http://ovi1rdu1p.bkt.clouddn.com/jb-like-track.png" alt="广外耕田轨迹截图">

这个杰宝形的轨迹最初还能在朋友圈上引得朋友一阵笑声，博得为数不多的几个 like 。然而，两三次故技重施之后，连那几个为数不多的 like 也全然消失，无迹可寻了。我便索性删掉朋友圈里的全部杰宝轨迹——与其看着空荡荡的点赞列表，不如忘记它，就像此事从未发生过。

<img id="del-moment" src="http://ovi1rdu1p.bkt.clouddn.com/del-moment.jpg" alt="删除朋友圈">

就算没有朋友圈可发，key 依旧不能停。去隔壁学校转圈依旧是每日必修。

某日深夜，我从中环往回走，瞥见前方一对依偎在一起的身影。我低头望回手机，同时加快了脚步，超过了他们。

突然，身后的男声一把叫住了我：

"Hey, Siri!"

我停下脚步。

“**你知道最近的宾馆在哪吗？**”

他手上的 Siri 替我回答了这个令人尴尬的问题：

“广州某点酒店，位于大学城某某路某号新天地某层，正在为您导航。”

## 0x02

这真令人悲伤。

**孑然一身地守护人类文明**，并不是一件能让自己免疫狗粮的理由。

然而还有一件令人悲伤的事情。每天这样夜晚出来 hack key 做多重，我注定**黑**不了。

```css
#Agent{
    background-color:black;
}
```

每做一次这样的多重都有 1,000 MUs，如果我想要拿到 Illminator Onyx Medal ，就势必要盖隔壁学校**四千次**。如果每天都能盖两次的话，那么……或许那个时候游戏早就关服了吧。

>1 Illminator Onyx Medal = 4,000 m CF

可是，如果想要开黑牌的话，还要同步清障，还要射拦截，还要做底边……想想就麻烦，不然还是……**等等，谁找我？**

>往【南沙】方向，你们看岛上需要连不?

>我最近会去一趟，可以拿个几十把回来。

>那，我拿你那开个牌？

>开。

## 0x03

我说，我想要黑牌，便有了这次黑牌。

然而，事情总是没有想象中的那么顺利。Draw Tool 一画， Cross Link 一开，满眼皆红，以挡住**东北-西南走向的腰**的 block link 为甚。

//iitc cross link with blur and red circles

果然还是要找人**清障**啊。

于是乎，在 Intel Map 上左看看，右看看，便找上了一个活跃的当地友军 SunSunLAW 。

> SunSunLAW 大佬，我们打算在大学城开黑牌，想请问一下【大石】附近能不能帮忙清一下障，或者哪些友军能够帮忙清障？要清的部分就是红圈标出来的那些 Portals 。

> 左边的红圈是 Sslian 去很方便，右手边是 DeadShip2003 活跃，还有近期有个很活跃的新人 WatanabeNan 。

> 感谢 SunSunLAW 引荐~

循着 SunSunLAW 给出的 ID 寻找下去，几位特工都欣然应允。

> Sslian /OK

> WatanabeNan 一定尽力而为

> DeadShip2003 好吧

另外，还有几条从博物馆射出的 link ，我找到了之前早有合作的 este36

> este36 好的，到时候我去博物馆待命吧

解决了东北-西南走向的 block links 之后，问题就像降低了一个维度——底边没有任何阻挡，而另一条腰上只有零星两三根或蓝或绿的 block link 。关于这几条零星的 block ，AprilDEF 决定**提前一天**前往亚运城清理 ENL link ，而其他部分，我联系了 Saaaage。

> Saaaage ok

至此，关于这次黑牌计划的清障工作已经完全做好了。嗯，**完全**。

## 0x04

我[掐指一算](http://www.fieldsizeestimator.com/)，发觉一层大概能有近 400k MUs，而岛上的 portals 能给我们盖近 40 层。换言之，这是一台能够带上四个人黑的**小轿车**。

> Field has a predicted 386472.7 MU.

有肉一起吃，有车一起上。我当即询问了岛上的几位小伙伴。

>龙妈：你想要黑牌吗。jpg

> Archernara 好诶,那我就留下了

> ikiwixx 似乎达成了共识/滑稽

> CourtingJuans 算上 juananss 一份

> ArielAxionL 广大还可以出两个人

> Howeverkcy 我还没黑

怎么就突然这么多人了？我一台核载四人的小轿车，怎么就有七个人想上车？乖乖，超载可是违法的。

> 《道路交通安全法》 第四十九条　机动车载人不得超过核定的人数，客运机动车不得违反规定载货。

为了避免交通事故，我找到了负责底边的 Slcio ，如果能够毒几轮，那小轿车就能进化成**大客车**。

> Slcio 找个人跟我来，一起摸了带key回去。我在这里等射底边，key足够的情况下可以毒多几轮。

这样情况就明朗了：在开牌之前，我先去一趟【南沙】摸key，并且带回岛上， Slcio 则留在【南沙】。在开黑牌的时候，我们每完成一轮， Slcio 就把底边毒掉，我们又可以重新再拉一次。三个po毒两轮的话，就有六倍于前的 MUs，也就是说，核载四人的小轿车进化成了**核载二十四人的大客车**。

这样一来，原本的半小时作业就拉长成了三个小时的大工程，附近的 ENL Agent 们肯定是不会坐看自己反复被盖的。

```javascript
var ENL = ingress.getElementsByClassName("ENL-Agent");
for (var i = 0; i < ENL.length; ++i) 
  ENL[i].addEventListener("coveredByBlueField", angry);
```

没办法，还得找人处理可能出现的 ENL block 。干脆就找开黑第一梯队的人，去它们可能出现的地方**蹲点**算了。
